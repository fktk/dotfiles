# Query Wiki Skill

このスキルは、`/wiki/` 内の各ディレクトリに散らばった知識を検索・統合し、ユーザーの問いに対する回答を生成するためのワークフローです。

## 使用手順

1. **実行:** このスキルを起動し、質問を入力する。
2. **回答:** AIが知識ベース全体を推論し、根拠となるWikiページを引用しながら回答する。

## ワークフロー (Agentic Steps)

1. **横断的検索:** 質問に関連する概念を、特定のディレクトリ（例: `/wiki/Finance/`）だけでなく、全ディレクトリから抽出する。
2. **統合的推論:** 分野が異なるページ（例: 投資手法とXRD解析の共通点）であっても、内容が関連すれば情報を統合して回答を構築する。
3. **正確な引用:** `[[ページ名]]` 形式でのリンクに加え、必要に応じてどのディレクトリにある情報かを明示する。
4. **不足情報の補完:** どのディレクトリに情報が不足しているかを報告し、補完すべき資料を提案する。

## ルール
- **ディレクトリに縛られない:** 物理的なフォルダ分けに関わらず、知識のつながりを重視して回答すること。
- **透明性:** 参照したページのパスやリンクを明示し、情報の出所を明確にすること。
