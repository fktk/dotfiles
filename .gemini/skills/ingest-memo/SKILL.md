# Ingest Memo Skill

このスキルは、`/raw/` フォルダ内の生データを読み込み、`/wiki/` フォルダへ構造化して取り込むためのワークフローです。

## 使用手順

1. **ファイルを配置:** `/raw/` フォルダに新しい資料（論文、記事、メモ等）を配置する。
2. **実行:** このスキルを起動し、対象ファイルを指定する。

## ワークフロー (Agentic Steps)

1. **分析:** `/raw/` 内の対象ファイルを読み込む。
2. **分類と配置:** 内容のトピックを判断し、適切なサブディレクトリ（例: `/wiki/Finance/`, `/wiki/Science/XRD/`）を特定、または作成する。
3. **要約・構造化:** 内容をMarkdown形式のWikiページに変換し、特定したディレクトリに保存する（既存ページとの関連付けを行う）。
4. **リンクの更新:** 関連するWikiページ（分野を跨ぐ場合も含む）にリンクを追加し、必要に応じて `Index.md` を更新する。
5. **追跡:** 取り込んだファイルには「処理済み」として記録を残すか、または `/raw/` からアーカイブフォルダへ移動する（運用による）。

## ルール
- `/raw/` は変更不可。
- Wikiへの取り込みは必ず `/wiki/` 内のファイルを更新/作成する形で行う。
- トピックごとにサブディレクトリを作成して整理を推奨するが、共通概念は `[[ページ名]]` でリンクを貼り、知識を結合させること。
